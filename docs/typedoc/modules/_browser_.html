External module "browser"

Browser interface

Bridge between the HTML form and the conversion facilities. It relies on <input> elements with the id values set to url, respec, publishDate, specStatus, addSectionLinks, and maxTocLevel.

This script must be converted to Javascript and the browserified to be used in an HTML page.

  • submit(event: Event): Promise<any>
    • Defined in browser.ts:27

    Generate the EPUB file. This is a wrapper around create_epub, creating the necessary arguments Arguments structure based on the incoming form data. The result is saved on the local disc, using the short name of the document.

    The method is set as an even handler for a submit button. The event argument is only used to prevent the default behavior of the button (i.e., to avoid reloading the page).
