r2epub — Converting HTML documents produced by Respec into EPUB 3.3

Typescript program to convert W3C documents, produced by ReSpec, to EPUB 3.3. The conversion can be done via:

  • A simple command line interface.
  • Using a Web Server API; an HTML interface to the service is available (using a service running on at the W3C)

See the full documentation for the details and usage examples.
